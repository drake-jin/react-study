## JSX 

### JSX의 장점 
 1. JSX는 컴파일링 되면서 최적화 되므로, 빠르다.
 2. Type-Safe 어떠한 연산도 정의되지 않은 결과를 내놓지 않는 것, 즉 예측 불가능한 결과를 나타내지 않는 것으로 하며, 컴파일링 과정에서 에러를 감지할 수 있다.
 3. HTML에 익숙하다면 JSX의 사용은 좀 더 명시적이고 빠르게 템플릿을 작성할 수 있다.  

 - JSX와 JS 소스코드적인 차이.
